**Дипломаная работа(backend)**
=====================
## Описание
Дипломная работа по курсу веб-разработки от Яндекс.Практикум. Бэкенд для сервиса по поиску фильмов по запросу и сохранения понравившихся фильмов в личном кабинете

## Стек
* Node.js
* Express.js
* MongoDB
* Mongoose
* Celebrate
* Joi
* Winston
* Helmet
* Bcryptjs
* Jsonwebtoken
* Nodemon
* Eslint
